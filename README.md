# Marketing Website - Astro & Tailwind CSS

A modern responsive marketing website built using Astro and Tailwind CSS.

## 🚀 Features

* Responsive Design
* Modern UI/UX
* Fast Performance
* Clean Code Structure
* Mobile Friendly Navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* Tailwind CSS
* Astro
* JavaScript

## 📂 Project Structure

```bash
project/
│── index.html
│── Style/
│   └── main.css
```

## ▶️ Getting Started

1. Clone the repository
2. Open the project folder
3. Run the project in your browser
